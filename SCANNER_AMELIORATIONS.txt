════════════════════════════════════════════════════════════════════════════════
   🎵 TYPES DE SONS
════════════════════════════════════════════════════════════════════════════════

✅ SUCCESS (playScanSound)
   • Fréquence : 600 Hz + 800 Hz (double note)
   • Durée : 0.15s + 0.15s
   • Usage : Scan réussi, validation réussie

⚠️ WARNING (playWarningSound)
   • Fréquence : 400 Hz
   • Durée : 0.2s
   • Usage : Colis déjà scanné, colis en transit

❌ ERROR (playErrorSound)
   • Fréquence : 200 Hz (grave)
   • Durée : 0.3s
   • Usage : Colis non trouvé

ℹ️ INFO
   • Fréquence : 500 Hz
   • Durée : 0.1s
   • Usage : Information générale

   🔄 FLUX D'UTILISATION
════════════════════════════════════════════════════════════════════════════════

1. SCAN D'UN NOUVEAU COLIS
   ────────────────────────
   [Scan QR] → [Son succès 🔊] → [Toast "Colis sélectionné"] → [Ajout au tracking]

2. RE-SCAN DU MÊME COLIS
   ────────────────────────
   [Scan QR] → [Son avertissement 🔊] → [PAS de toast] → [Déjà dans le tracking]

3. SCAN D'UN COLIS DÉJÀ RÉCEPTIONNÉ
   ────────────────────────────────────
   [Scan QR] → [Son avertissement 🔊] → [Toast "Déjà réceptionné" UNE FOIS] → [Ajout au tracking]

4. SCAN D'UN COLIS EN TRANSIT
   ────────────────────────────
   [Scan QR] → [Son avertissement 🔊] → [Toast "En transit" UNE FOIS] → [Ajout au tracking]

5. SCAN D'UN COLIS INEXISTANT
   ────────────────────────────
   [Scan QR] → [Son erreur 🔊] → [Toast "Non trouvé" TOUJOURS] → [Pas de tracking]

6. VALIDATION DE RÉCEPTION
   ────────────────────────────
   [Clic Valider] → [Réception] → [Son succès 🔊] → [Réinitialisation tracking]

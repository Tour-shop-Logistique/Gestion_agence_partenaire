FONCTIONNEMENT :
===============

Lorsque l'utilisateur sélectionne le type "LD" (SIMPLE) :

1. Le système récupère tous les tarifs simples configurés par l'agence
2. Parcourt toutes les zones et extrait leurs tableaux de pays
3. Dédoublonne et trie les pays par ordre alphabétique
4. Affiche la liste des pays dans un menu déroulant avec recherche
5. Quand un pays est sélectionné :
   - Le pays de destination est automatiquement rempli
   - Le tarif et la zone correspondants sont stockés pour la simulation
   - La ville de destination est vidée pour saisie manuelle

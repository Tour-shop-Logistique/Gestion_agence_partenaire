# 🚀 Guide rapide : Nouvelle création d'expédition

## 📱 Les 4 étapes en un coup d'œil

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   ÉTAPE 1   │ →  │   ÉTAPE 2   │ →  │   ÉTAPE 3   │ →  │   ÉTAPE 4   │
│   Trajet    │    │    Colis    │    │   Clients   │    │  Paiement   │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

---

## 🎯 ÉTAPE 1 : Choisir le trajet

### Que faire ?
✅ Sélectionner le **type d'expédition** (LD, DHD Aérien, DHD Maritime, etc.)
✅ Choisir un **trajet disponible** (si applicable)
✅ Vérifier le **pays et ville de destination**
✅ Cocher les options : Livraison à domicile / Paiement à crédit

### Champs obligatoires (bordure jaune)
- Type d'expédition
- Pays destination
- Ville destination

### Bouton suivant
🔵 **"Suivant : Enregistrer les colis"**

---

## 📦 ÉTAPE 2 : Enregistrer les colis

### Que faire ?
✅ Pour chaque colis :
   - Saisir la **désignation** (ex: "Carton de vêtements")
   - Entrer le **poids en kg** (obligatoire)
   - Ajouter les dimensions (optionnel)
   - Indiquer les frais d'emballage
   - Ajouter des articles si besoin

✅ Ajouter d'autres colis avec le bouton **"+ Ajouter un colis"**

✅ Cliquer sur **"Calculer le tarif"** (ou Ctrl+S)

### Champs obligatoires (bordure jaune)
- Désignation du colis
- Poids du colis

### Résultat
📊 Un panneau vert apparaît à droite avec :
- Frais d'expédition
- Frais d'emballage
- **TOTAL à encaisser**

### Bouton suivant
🟢 **"Suivant : Identifier les clients"**

---

## 👥 ÉTAPE 3 : Identifier les clients

### Que faire ?
✅ **Expéditeur** (celui qui envoie)
   - Nom complet
   - Téléphone
   - Email (optionnel)
   - Adresse (optionnel)

✅ **Destinataire** (celui qui reçoit)
   - Nom complet
   - Téléphone
   - Email (optionnel)
   - Adresse (optionnel)
   - Code postal (optionnel)

### Champs obligatoires (bordure jaune)
- Nom et téléphone de l'expéditeur
- Nom et téléphone du destinataire

### Bouton suivant
🔵 **"Suivant : Encaisser et valider"**

---

## 💰 ÉTAPE 4 : Encaisser et valider

### Que faire ?
✅ Vérifier le **récapitulatif** de l'expédition

✅ Choisir le **mode de paiement** :
   - 💵 **Espèces**
   - 📱 **Mobile Money** (saisir la référence)
   - 🏦 **Virement bancaire**

✅ Si paiement à crédit : Aucun encaissement

### Panneau de droite
💵 Affiche le **montant total à encaisser** en gros

### Bouton final
🟢 **"Valider l'expédition"** (ou Ctrl+Enter)

### Après validation
✅ Modal de succès avec options d'impression
✅ Redirection vers la liste des expéditions

---

## ⌨️ Raccourcis clavier

| Raccourci | Action |
|-----------|--------|
| `Ctrl + S` | Calculer le tarif (étape 2) |
| `Ctrl + Enter` | Valider l'expédition (étape 4) |
| `Ctrl + →` | Passer à l'étape suivante |
| `Ctrl + ←` | Revenir à l'étape précédente |

---

## 🎨 Comprendre les couleurs

### Bordures des champs
- 🟡 **Jaune/Ambre** : Champ obligatoire à remplir
- 🟢 **Vert** : Champ rempli correctement
- ⚪ **Gris** : Champ optionnel vide
- 🔒 **Gris clair** : Champ désactivé

### Indicateur d'étapes
- ⚫ **Noir** : Étape actuelle
- ✅ **Vert avec ✓** : Étape complétée
- ⚪ **Gris** : Étape à venir

---

## 💡 Astuces pratiques

### 1. Navigation flexible
Vous pouvez **cliquer sur les étapes précédentes** pour revenir en arrière et modifier des informations.

### 2. Validation automatique
Impossible de passer à l'étape suivante si les champs obligatoires ne sont pas remplis.

### 3. Panneau de tarification
À l'étape 2, le panneau de tarification reste **collé à droite** pour toujours voir le montant.

### 4. Multi-colis
Ajoutez autant de colis que nécessaire. Le poids total et les frais sont calculés automatiquement.

### 5. Paiement à crédit
Si vous cochez "Paiement à crédit" à l'étape 1, vous n'aurez pas à choisir de mode de paiement à l'étape 4.

---

## ❓ Questions fréquentes

### Puis-je modifier une étape après l'avoir validée ?
✅ **Oui !** Cliquez simplement sur l'étape dans l'indicateur en haut.

### Que se passe-t-il si j'oublie un champ obligatoire ?
⚠️ Le bouton "Suivant" sera grisé et un message vous indiquera ce qui manque.

### Comment ajouter plusieurs colis ?
➕ Utilisez le bouton **"+ Ajouter un colis"** à l'étape 2.

### Dois-je calculer le tarif à chaque fois ?
✅ **Oui**, c'est obligatoire pour passer à l'étape 3. Utilisez Ctrl+S pour aller plus vite.

### Puis-je annuler la création ?
🔙 Utilisez le bouton "Retour" ou fermez simplement la page.

---

## 🆘 En cas de problème

### Le bouton "Suivant" ne fonctionne pas
→ Vérifiez que tous les champs avec bordure jaune sont remplis

### Le calcul de tarif échoue
→ Vérifiez que le poids et la destination sont corrects

### L'impression ne fonctionne pas
→ Vérifiez les paramètres de votre navigateur

### Autre problème
→ Contactez le support technique avec une capture d'écran

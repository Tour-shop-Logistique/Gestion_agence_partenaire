MÉTHODE 1 : Scan automatique (recommandé)
──────────────────────────────────────────
1. Cliquez sur "Scanner"
2. Autorisez l'accès à la caméra
3. Placez le QR code devant la caméra
4. Le colis est automatiquement sélectionné !

MÉTHODE 2 : Saisie manuelle (si caméra indisponible)
─────────────────────────────────────────────────────
1. Cliquez sur "Scanner"
2. Utilisez le champ de saisie en bas
3. Tapez le code colis (ex: COL001)
4. Cliquez "Valider"

═══════════════════════════════════════════════════════════════
  📱 PERMISSIONS CAMÉRA
═══════════════════════════════════════════════════════════════

Première utilisation :
  • Le navigateur demandera l'autorisation
  • Cliquez sur "Autoriser"
  • Le scanner démarre automatiquement

Si refusé par erreur :
  • Cliquez sur le cadenas (barre d'adresse)
  • Autorisez l'accès à la caméra
  • Rechargez la page

═══════════════════════════════════════════════════════════════
  🔍 RECHERCHE INTELLIGENTE
═══════════════════════════════════════════════════════════════

Le scanner cherche le colis de 3 façons :

1. Code_colis exact : COL001
2. Code_colis partiel : Si QR contient plus d'infos
3. ID expédition : Pour compatibilité anciens QR

Résultat :
  ✓ Colis trouvé → Sélection + Scroll + Message succès
  ✗ Colis non trouvé → Message d'erreur

═══════════════════════════════════════════════════════════════
  📦 FORMAT DU QR CODE
═══════════════════════════════════════════════════════════════

Nouveau format (reçus thermiques) :
  QR Code = Code colis (COL001)

Ancien format (compatibilité) :
  QR Code = ID expédition (123)

Le scanner gère les deux formats automatiquement !

═══════════════════════════════════════════════════════════════
  🎉 SCÉNARIO DE RÉCEPTION RAPIDE
═══════════════════════════════════════════════════════════════

Réception de 10 colis en moins de 2 minutes :

1. Ouvrir "Colis à réceptionner"
2. Cliquer "Scanner"
3. Scanner le 1er QR code → Sélectionné !
4. Fermer le scanner
5. Cliquer "Scanner" à nouveau
6. Scanner le 2ème QR code → Sélectionné !
7. Répéter pour les 8 autres colis
8. Cliquer "Réceptionner la sélection"
9. Tous les colis sont réceptionnés !

Gain de temps : 70% plus rapide qu'avant !

═══════════════════════════════════════════════════════════════
  🐛 DÉPANNAGE
═══════════════════════════════════════════════════════════════

Problème : Caméra ne s'ouvre pas
Solution :
  • Vérifiez les permissions
  • Utilisez HTTPS (requis)
  • Essayez un autre navigateur
  • Utilisez la saisie manuelle

Problème : QR code non détecté
Solution :
  • Améliorez l'éclairage
  • Rapprochez/éloignez le QR code
  • Assurez-vous que le QR est net
  • Utilisez la saisie manuelle

Problème : Colis non trouvé
Solution :
  • Vérifiez que le colis est dans la liste
  • Actualisez la page
  • Vérifiez le code manuellement

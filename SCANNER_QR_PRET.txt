  🎯 SCÉNARIO D'UTILISATION
═══════════════════════════════════════════════════════════════

Réception rapide de plusieurs colis :

1. Cliquez sur "Scanner"
2. Regardez le code sur le reçu papier
3. Tapez le code dans le champ
4. Cliquez "Valider"
5. Répétez pour chaque colis
6. Cliquez "Réceptionner la sélection"

Gain de temps : Plus besoin de chercher dans la liste !

═══════════════════════════════════════════════════════════════
  📱 SCAN AUTOMATIQUE (OPTIONNEL)
═══════════════════════════════════════════════════════════════

Pour activer le scan automatique avec la caméra :

1. Installez la bibliothèque :
   npm install html5-qrcode

2. Redémarrez le serveur :
   Ctrl + C puis npm run dev

3. Le scan automatique sera activé !

═══════════════════════════════════════════════════════════════
  🔍 RECHERCHE INTELLIGENTE
═══════════════════════════════════════════════════════════════

Le scanner cherche le colis de 3 façons :

1. Par code_colis exact (COL001)
2. Par code_colis partiel
3. Par ID d'expédition (compatibilité)

Si le colis est trouvé :
  ✓ Il est sélectionné automatiquement
  ✓ La page scroll vers lui
  ✓ Message de succès affiché

Si le colis n'est pas trouvé :
  ✗ Message d'erreur
  ✗ Vérifiez le code ou actualisez la page

## 📱 Comment Utiliser

### Navigation Normale

1. **Première visite du dashboard**
   - Vous verrez le loader circulaire (~2 secondes)
   - Les données s'affichent
   - L'horodatage apparaît : "Mis à jour 14:32"

2. **Naviguer ailleurs puis revenir rapidement**
   - Allez sur Expéditions, Colis, etc.
   - Revenez au Dashboard (< 30 secondes)
   - ⚡ Affichage instantané !

3. **Revenir après un moment**
   - Si vous revenez après 30 secondes
   - Le loader s'affiche à nouveau
   - Les données sont actualisées

### Actualisation Manuelle

**Quand l'utiliser ?**
- Vous voulez voir les toutes dernières données
- Vous savez qu'il y a eu des changements

**Comment faire ?**
1. Cliquez sur le bouton [↻] en haut à gauche
2. Vous verrez "Actualisation..." apparaître
3. Les données restent visibles pendant l'actualisation
4. L'horodatage se met à jour après

## ❓ Questions Fréquentes

### Q1 : Pourquoi le dashboard ne se recharge pas toujours ?
**R** : C'est normal ! Si vous revenez rapidement (< 30 secondes), les données en cache sont affichées instantanément. C'est une fonctionnalité, pas un bug !

### Q2 : Comment savoir si mes données sont à jour ?
**R** : Regardez l'horodatage "Mis à jour HH:MM" en haut. Si c'est récent (< 30 secondes), vos données sont fraîches.

### Q3 : Comment forcer une actualisation ?
**R** : Cliquez sur le bouton [↻] en haut à gauche. Les données seront rechargées depuis le serveur.

### Q4 : Le loader circulaire est normal ?
**R** : Oui ! C'est le nouveau design professionnel. Il s'affiche seulement au premier chargement ou après 30 secondes d'inactivité.

### Q5 : Mes données sont-elles toujours à jour ?
**R** : Oui ! Le système actualise automatiquement si les données ont plus de 30 secondes. Vous pouvez aussi forcer l'actualisation avec [↻].

## 🐛 Problèmes Courants

### Problème : Le dashboard ne s'affiche pas
**Solution** :
1. Vérifiez votre connexion internet
2. Actualisez la page (F5)
3. Videz le cache du navigateur (Ctrl+Shift+Delete)

### Problème : L'horodatage ne se met pas à jour
**Solution** :
1. Cliquez sur [↻] pour forcer l'actualisation
2. Vérifiez que le serveur répond

### Problème : Le loader reste affiché
**Solution** :
1. Vérifiez votre connexion internet
2. Actualisez la page (F5)
3. Contactez le support si le problème persiste

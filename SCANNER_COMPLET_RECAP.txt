════════════════════════════════════════════════════════════════════════════════
   🔧 MÉCANISMES TECHNIQUES
════════════════════════════════════════════════════════════════════════════════

1. DEBOUNCE DU SCANNER (QRScanner.jsx)
   ────────────────────────────────────────────────────────────────────────
   lastScannedRef = {
       code: "COL001",
       timestamp: 1720700000000
   }
   
   À chaque scan :
   - Comparer le code avec lastScannedRef.code
   - Comparer le timestamp (now - lastScannedRef.timestamp)
   - Si même code ET < 2000ms → IGNORER
   - Sinon → ACCEPTER et mettre à jour lastScannedRef

2. TRACKING DES MESSAGES (ColisAReceptionner / ReceptionColis)
   ────────────────────────────────────────────────────────────────────────
   scannedCodesRef = Set(["COL001", "COL002", "COL999"])
   
   À chaque scan :
   - Vérifier si code dans scannedCodesRef
   - Si présent → Ne pas afficher de toast
   - Si absent → Afficher toast ET ajouter au Set
   
   Réinitialisation après validation réussie

3. VÉRIFICATION ANTI-DOUBLONS SÉLECTION
   ────────────────────────────────────────────────────────────────────────
   selectedCodes = ["COL001", "COL002"]
   
   Avant ajout :
   - if (!selectedCodes.includes(code)) → Ajouter
   - else → Message "déjà sélectionné"

4. GÉNÉRATION DES SONS (soundNotification.js)
   ────────────────────────────────────────────────────────────────────────
   Web Audio API :
   - Création d'un AudioContext unique
   - Oscillateurs pour générer les fréquences
   - GainNode pour contrôler le volume
   - Différentes fréquences selon le type :
     * Succès : 600 Hz + 800 Hz (2 notes)
     * Avertissement : 400 Hz
     * Erreur : 200 Hz (grave)

# 📚 Plan de migration et formation - Nouvelle création d'expédition

## 🎯 Objectifs

1. **Former** tous les agents à la nouvelle interface en 4 étapes
2. **Migrer** en douceur sans perturber l'activité
3. **Assurer** un support continu pendant la transition
4. **Mesurer** l'impact et les améliorations

---

## 📅 Planning de déploiement

### Semaine 1 : Préparation
- **Jour 1-2** : Tests internes par l'équipe technique
- **Jour 3-4** : Sélection des agents pilotes (2-3 personnes)
- **Jour 5** : Formation des agents pilotes

### Semaine 2 : Phase pilote
- **Jour 1-5** : Agents pilotes utilisent le nouveau système
- **Collecte** : Feedback quotidien
- **Ajustements** : Corrections mineures si nécessaire

### Semaine 3 : Déploiement progressif
- **Jour 1** : Formation groupe 1 (30% des agents)
- **Jour 2** : Formation groupe 2 (30% des agents)
- **Jour 3** : Formation groupe 3 (40% des agents)
- **Jour 4-5** : Support intensif

### Semaine 4 : Consolidation
- **Jour 1-3** : Monitoring et support
- **Jour 4** : Bilan intermédiaire
- **Jour 5** : Ajustements finaux

---

## 👥 Formation des agents

### Format de la formation

#### Durée : 30 minutes par session
- **10 min** : Présentation du nouveau workflow
- **15 min** : Démonstration en direct
- **5 min** : Questions / Réponses

#### Groupes : 5-8 agents maximum
- Permet l'interaction
- Facilite les questions
- Meilleure attention

---

### 📖 Contenu de la formation

#### 1. Introduction (2 minutes)
```
"Bonjour à tous ! Aujourd'hui nous allons découvrir la nouvelle 
interface de création d'expédition. Elle passe de 2 à 4 étapes 
pour plus de clarté et de rapidité."
```

**Points clés :**
- Pourquoi le changement ?
- Quels bénéfices pour vous ?
- Pas d'inquiétude, c'est plus simple !

---

#### 2. Vue d'ensemble (3 minutes)

**Montrer l'indicateur d'étapes :**
```
[1 Trajet] → [2 Colis] → [3 Clients] → [4 Paiement]
```

**Expliquer :**
- 4 étapes au lieu de 2
- Chaque étape a un objectif clair
- Impossible d'oublier des informations
- Navigation flexible (retour possible)

---

#### 3. Démonstration pratique (15 minutes)

##### Exemple 1 : Expédition DHD Aérien simple

**ÉTAPE 1 : Trajet (2 min)**
```
1. Sélectionner "DHD AERIEN"
2. Choisir le trajet "Abidjan-Paris"
3. Vérifier destination : France, Paris
4. Cocher "Livraison à domicile"
5. Cliquer "Suivant"
```

**Montrer :**
- Les champs obligatoires (bordure jaune)
- Les champs remplis (bordure verte)
- Le bouton "Suivant" qui s'active

---

**ÉTAPE 2 : Colis (4 min)**
```
1. Désignation : "Carton de vêtements"
2. Poids : 15 kg
3. Dimensions : 50 x 40 x 30 cm
4. Emballage : 2000 FCFA
5. Ajouter articles (optionnel)
6. Cliquer "Calculer le tarif" (Ctrl+S)
```

**Montrer :**
- Le panneau de tarification qui apparaît
- Le montant total affiché
- Le bouton "Suivant" qui s'active après simulation

**Astuce :** "Utilisez Ctrl+S pour calculer plus vite !"

---

**ÉTAPE 3 : Clients (4 min)**
```
EXPÉDITEUR
- Nom : Jean Kouassi
- Téléphone : +225 07 12 34 56 78
- Email : jean@example.com (optionnel)

DESTINATAIRE
- Nom : Marie Dupont
- Téléphone : +33 6 12 34 56 78
- Email : marie@example.com (optionnel)
```

**Montrer :**
- La séparation visuelle (badges bleu/violet)
- Les champs obligatoires vs optionnels
- Le bouton "Suivant" qui s'active

---

**ÉTAPE 4 : Paiement (3 min)**
```
1. Vérifier le récapitulatif
2. Choisir "Espèces"
3. Vérifier le montant total (panneau de droite)
4. Cliquer "Valider l'expédition" (Ctrl+Enter)
```

**Montrer :**
- Le récapitulatif complet
- Les 3 modes de paiement
- Le panneau avec le montant en gros
- La modal de succès avec impression

---

##### Exemple 2 : Expédition avec retour arrière (2 min)

**Scénario :**
```
"Oups, j'ai oublié de cocher 'Livraison à domicile' !"
```

**Solution :**
```
1. Cliquer sur "1 Trajet" dans l'indicateur
2. Cocher l'option
3. Cliquer sur "2 Colis" pour revenir
4. Continuer normalement
```

**Message :** "Vous pouvez revenir en arrière à tout moment !"

---

#### 4. Raccourcis clavier (2 minutes)

**Afficher le tableau :**
```
┌─────────────────┬──────────────────────────────┐
│ Ctrl + S        │ Calculer le tarif (étape 2)  │
│ Ctrl + Enter    │ Valider (étape 4)            │
│ Ctrl + →        │ Étape suivante               │
│ Ctrl + ←        │ Étape précédente             │
└─────────────────┴──────────────────────────────┘
```

**Conseil :** "Utilisez-les pour gagner du temps !"

---

#### 5. Codes couleur (2 minutes)

**Montrer les bordures :**
```
🟡 Jaune  → Champ obligatoire à remplir
🟢 Vert   → Champ rempli correctement
⚪ Gris   → Champ optionnel vide
🔒 Gris clair → Champ désactivé
```

**Conseil :** "Suivez les couleurs, elles vous guident !"

---

#### 6. Questions fréquentes (3 minutes)

**Q1 : "Puis-je revenir en arrière ?"**
✅ Oui ! Cliquez sur l'étape dans l'indicateur.

**Q2 : "Que faire si j'oublie un champ ?"**
✅ Le bouton "Suivant" sera grisé et un message vous le dira.

**Q3 : "Comment ajouter plusieurs colis ?"**
✅ Bouton "+ Ajouter un colis" à l'étape 2.

**Q4 : "Dois-je toujours calculer le tarif ?"**
✅ Oui, c'est obligatoire pour passer à l'étape 3.

**Q5 : "Et si je préfère l'ancienne version ?"**
✅ Testez la nouvelle pendant 1 semaine, vous verrez la différence !

---

#### 7. Exercice pratique (5 minutes)

Colis : 1 carton de 20 kg

Destinataire : Nom fictif
Paiement : Espèces

**Observer et aider si nécessaire.**

## 🆘 Gestion des problèmes

### Problème : Agent résistant au changement
**Solution :**
1. Écouter ses préoccupations
2. Montrer les bénéfices concrets
3. Accompagnement personnalisé
4. Période d'adaptation plus longue

### Problème : Bug technique
**Solution :**
1. Documenter précisément
2. Contacter le support technique
3. Workaround temporaire si possible
4. Communication transparente

### Problème : Confusion sur une étape
**Solution :**
1. Revoir la formation
2. Améliorer la documentation
3. Ajouter des tooltips si nécessaire
4. Vidéo explicative ciblée

═══════════════════════════════════════════════════════════════
  📋 UTILISATION
═══════════════════════════════════════════════════════════════

1. Allez sur "Colis à réceptionner"
2. Cliquez sur le bouton "Scanner"
3. Autorisez l'accès à la caméra
4. Scannez le QR code du reçu
5. Le colis est automatiquement sélectionné !

Alternative : Saisie manuelle du code colis

═══════════════════════════════════════════════════════════════
  📱 FORMAT DU QR CODE
═══════════════════════════════════════════════════════════════

AVANT : QR code = ID expédition (123)
APRÈS : QR code = Code colis (COL001)

Le scanner cherche le colis de 3 façons :
  1. Par code_colis exact
  2. Par code_colis partiel
  3. Par ID d'expédition (compatibilité)

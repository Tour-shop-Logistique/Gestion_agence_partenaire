## 🐛 Dépannage

### La page ne s'affiche pas
➡️ Vérifier la console navigateur
➡️ S'assurer que TailwindCSS est installé

### Les animations ne sont pas fluides
➡️ Vérifier la configuration Tailwind
➡️ S'assurer que les classes sont incluses

### Les filtres ne fonctionnent pas
➡️ Vérifier que les données sont chargées
➡️ Console navigateur pour erreurs
